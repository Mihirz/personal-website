# mihir.sh — personal website

Zero-dependency single-page site. Open `index.html` in a browser — nothing to build.

## What's inside
- `index.html` — structure/content
- `styles.css` — theme, layout, animations
- `script.js` — canvas effects & interactivity (all procedural, no images)

## Bells & whistles
- Live neural-network background (nodes attract your cursor; magenta signal pulses)
- Pixel-art self-portrait (canvas-rendered, with animated scan line)
- Pixel-art worm that follows your scroll — nod to the C. elegans project
- Typewriter hero, glitched name, scan-line overlay, custom cursor
- Per-project pixel scenes (worm / robot / brain / gears)
- Terminal-styled contact card
- Konami code easter egg (↑↑↓↓←→←→BA) — try it
